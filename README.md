
# Spotify-Clone

A fully responsive music player clone inspired by Spotify's popular look and feel, with a user-friendly interface. 



## Project Overview

This project is designed to mimic the user interface and core functionalities of Spotify. Through this, users will experience song play, pause, volume control, and playlist navigation. This is an attempt to showcase my front-end skills.

## Installation

Clone the repo and install dependencies

```bash
  git clone https://github.com/emon-ahmed-60/Spotify-Clone.git
  cd Spotify-Clone
  npm install
```
Run the application:

```bash
  npm run dev
```
